# Customer-Churn-Probability-Analysis
Müşteri terk olasılığının istatistiksel modeller ve makine öğrenmesi ile analizi
# Müşteri Kayıp (Churn) Olasılık Analizi

Bu proje, bir telekomünikasyon şirketindeki müşteri terk oranlarını analiz etmek ve 
olasılık modelleri kullanarak hangi müşterilerin ayrılma riskinin yüksek olduğunu tahmin etmek amacıyla hazırlanmıştır.

## Kullanılan Teknolojiler
* Python (Pandas, NumPy, Scikit-learn)
* Veri Görselleştirme (Seaborn, Matplotlib)
* İstatistiksel Modelleme (Logistic Regression)

## Veri Seti Hakkında
Kaggle Telco Churn veri seti kullanılmıştır.
